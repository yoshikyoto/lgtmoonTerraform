## はじめに

tfenv の利用を推奨します。

Mac の場合は brew でインストールできます。

```sh
brew install tfenv
```

tfenv を使って terraform をインストールします。 
`.terraform-version` に書いてあるバージョンと、同じバージョンの terraform がインストールされます。

```sh
rfenv install
terraform --version                                                                                          [master]
```
